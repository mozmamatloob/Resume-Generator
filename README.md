Proud to share my latest project: AI-Powered Career Suite! 🚀

I’m excited to announce the completion of my AI Career Suite, a comprehensive tool designed to streamline the job application process using Generative AI. 🤖💼
Applying for jobs can be overwhelming, so I built this platform to help candidates create high-quality, ATS-friendly documents in seconds.

Key Features:
Resume Generator: Creates professional resumes with a clean, standardized header and bold, structured sections for better readability.
ATS Optimizer: Tailors existing resumes to specific Job Descriptions, ensuring keywords and formatting align with what recruiters are looking for.
Dynamic Cover Letter Writer: Generates personalized cover letters where users can input custom dates and company addresses for a professional touch.
Smart Download & Copy: Users can download their results as formatted HTML files to preserve bolding and clickable links, or simply copy the plain text to their clipboard.
Technical Stack:
Frontend: HTML5, CSS3, JavaScript (with marked.js for real-time Markdown rendering).
Backend: FastAPI (Python).
AI Engine: Google Gemini API (1.5 Flash).

This project helped me dive deep into Prompt Engineering, API Integration, and creating a seamless User Experience (UX).
